# mcp-stocklake

`mcp-stocklake` is the agent-facing MCP interface used by OpenClaw to interact with platform data and capabilities.

## Scope (scaffold phase)

- Define MCP server boundaries for stock platform operations
- Mediate safe agent access to datalake-backed capabilities
- Keep business logic deferred while contracts stabilize

## Planned structure

- `tests/` for MCP contract and handler tests
- server implementation modules to be added later
